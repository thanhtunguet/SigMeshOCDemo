# SigMeshOCDemo

Sig mesh demo project on iOS. Written in Objective-C

# Clone repository

```sh
git clone https://github.com/thanhtunguet/SigMeshOCDemo.git
# CD to project directory
cd SigMeshOCDemo
# Clone submodule (TelinkSigMeshLib)
git submodule update --init
```
